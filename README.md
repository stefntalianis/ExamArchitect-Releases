- ExamArchitect

Project Overview: Academic Assessment Tool
A high-performance desktop application developed for an academic faculty to streamline the exam creation process. This tool allows educators to generate, manage, and customize complex academic exams with a focus on data integrity and user-friendly design.

- Technologies Used

Frontend: React 


Desktop Framework: Electron 


Database: SQLite


Architecture: Node.js backend integration

- Key Features

Bespoke Generation: Automated exam creation based on custom parameters.


Advanced Customization: Dedicated options for faculty to tailor exam formats and questions to specific course needs.


Local Data Storage: Secure, offline data handling using SQLite for sensitive academic data.


Scalable Architecture: Built to allow for continuous technical updates and feature expansions.
